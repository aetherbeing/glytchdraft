# GlytchDraft 🪩
*The Co-Evolution Engine*

A spatial-media platform where players explore, claim, and co-create a glitch-saturated world—side by side with autonomous AI companions.

## Quick Start
1. \cd frontend && npm install\
2. Set up Supabase project
3. Configure environment variables
4. \
pm run dev\

## Structure
- \/frontend\ - React + Three.js spatial interface
- \/backend\ - Supabase functions + AI orchestration  
- \/lore\ - The 12 Orders, companion personalities, world data
- \/docs\ - Technical and design documentation

Let's make the first digital myth that laughs with the player and evolves with the machine.
