## Status Legend

| Status | Meaning |
|---|---|
| verified | Supported by a current repository artifact or official external source. |
| provisionally supported | Supported by repository evidence, but tied to candidate, stale, narrow, or not-yet-frozen artifacts. |
| planned | Intended future capability or deliverable; not evidence yet. |
| unsupported | Do not use externally until evidence is produced. |
| prohibited | Do not use. Claim is overbroad, misleading, legally risky, or outside Phase 1. |
| expired | Previously true or plausible but no longer current without re-verification. |
| needs review | Evidence exists but requires human/legal/technical review before use. |
