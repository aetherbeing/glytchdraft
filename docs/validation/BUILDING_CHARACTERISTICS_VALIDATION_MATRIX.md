---

## Status Controlled Vocabulary

| Code | Meaning |
|------|---------|
| `VERIFIED` | Code, schema, and test confirm the field is produced and correct in range |
| `PARTIALLY_VERIFIED` | Code confirms production; semantics or plausibility not independently tested |
| `AMBIGUOUS` | Field name or meaning is unclear or contradicted by multiple sources |
| `UNVALIDATED` | Field is produced but no automated test confirms its correctness |
| `NOT_IMPLEMENTED` | Field appears in schema or roadmap but no production code emits it |
| `DEPRECATED` | Field existed in prior pipeline versions; legacy code paths only |

## Confidence Controlled Vocabulary

| Code | Meaning |
|------|---------|
| `HIGH` | Units, computation, and CRS are all confirmed by code and cross-checked |
| `MEDIUM` | Main computation confirmed; edge cases or units inferred |
| `LOW` | Field is emitted but semantics rely on naming conventions or docs |
| `UNKNOWN` | Cannot determine confidence from current codebase evidence |
