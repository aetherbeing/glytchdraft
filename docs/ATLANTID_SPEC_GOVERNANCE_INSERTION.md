## Release Contract Governance

This specification is governed by `docs/ATLANTID_RELEASE_CONTRACT_GOVERNANCE.md`.

Contract compliance is a binary gate that precedes substantive review.

The release captain has no authority to waive a deviation.

Routine deviations are automatically enforced through bounded remediation without interrupting Charles.

Charles is interrupted only for genuine operator-level decisions involving substantial work loss, meaningful new cost or external commitment, material scope change, strategically different compliant paths, conflict with a later explicit instruction, absence of a compliant remediation path, or consideration of a waiver.

Safety- and provenance-critical gaps default to enforcement.

If no genuine operator-level tradeoff exists, STOP-AND-DECIDE must not be issued.
